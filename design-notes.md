# arah ui/ux d'fie catering

## visual direction
hangat, elegan, premium, dan homey. halaman dibuat seperti boutique catering: banyak ruang napas, foto/visual makanan dominan, warna cream sebagai base, deep green sebagai identitas premium, sage/gold sebagai aksen, dan warm brown untuk nuansa rumahan.

## design system
- base: #fbf3e6
- surface: #fffaf1
- primary: #23452e
- secondary: #6f8b63
- accent: #c49a4b
- text accent: #5b3c27
- radius: 18-32px
- shadow: soft, hangat, tidak berat
- heading: playfair display/georgia
- body: inter/arial

## section flow
navbar sticky -> hero -> trust/value -> menu favorit -> signature product -> galeri -> story -> testimoni -> faq -> kontak/final cta -> footer.

## copywriting utama
headline: catering hangat dengan rasa yang dibuat sepenuh hati.
cta: order via whatsapp.
brand story: taste with heart, fokus rasa, perhatian, detail, dan ketulusan.

## catatan data belum tersedia
prd belum berisi nomor whatsapp resmi, instagram resmi, alamat/maps, foto asli, harga asli, dan testimoni asli. frontend dibuat siap produksi secara struktur, tetapi data kontak/foto/testimoni perlu diganti sebelum publish final.
